Preguntas frecuentes

¿Puedo pedir información sobre varios productos?

Sí. Puedes indicar si deseas comparar el robot, módulos, paquetes, membresía o acceso digital.

¿El formulario envía datos reales?

No. Esta versión es demostrativa y está lista para conectarse después a un backend o servicio de correo.

¿Qué productos tienen modelo 3D?

Únicamente MEERA Rover, módulo de básquet robótico y módulo de circuito de carrera.
